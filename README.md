# Taleem Plus 🎓

**Taleem Plus** is a premium, modern AI-powered EdTech platform designed for local academy management. It provides a futuristic and highly interactive user experience while being optimized for fully offline usage.

## ✨ Features

- **Futuristic Splash Screen**: Animated gradient background with glowing logo, floating particles, and progress indicators.
- **Glassmorphic Login**: Secure and sleek authentication interface with animated input fields and password visibility toggles.
- **Multi-Role Registration**: Specialized sign-up flows for:
  - 👨‍🏫 **Teachers**: Manage classes and assessments.
  - 👨‍🎓 **Students**: Track learning progress.
  - 👪 **Parents**: Monitor child performance.
  - ⚙️ **Admins**: System administration and workspace management.
- **Modern Aesthetic**: Deep teal theme with cyan glow effects and smooth transitions.
- **Fully Offline**: Designed to work in environments with limited connectivity.

## 🚀 Getting Started

### Prerequisites

- Flutter SDK: `^3.12.0`
- Android SDK with NDK support.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/taleemplus.git
   ```
2. Navigate to the project directory:
   ```bash
   cd taleemplus
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```

## 🛠️ Build Information

To generate a release APK for Android:

```bash
flutter build apk --release
```

The output will be located at `build/app/outputs/flutter-apk/app-release.apk`.
